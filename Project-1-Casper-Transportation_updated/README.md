# Casper Transportation & Infrastructure Map

## Overview

This project was created in QGIS using OpenStreetMap data to create a transportation and infrastructure map of Casper, Wyoming.

## Skills Demonstrated

- GIS Mapping
- Cartographic Design
- Data Visualization
- Map Layout Creation
